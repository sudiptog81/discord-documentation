Troubleshooting

My bot won't connect!

Make sure you have properly set up the .env file to contain a DISCORD_TOKEN variable set to your bot's token.

I can't get my bot in my server

Make sure you are generating the invite link correctly, and that you have "Manage Server" permissions for the server you are trying to add the bot to.

My bot isn't responding to a command!

Make sure your bot has the appropriate permissions in the server. To verify if a permission error exists, try granting the bot the Administrator permission. If the command starts working, then your permissions are mis-configured.

Other Issues?

If you are having another issue, you can create a bug report or in our Discord server.
